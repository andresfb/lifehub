# LifeHub

A Personal Everything App

## Functionality

1. File collecting:
    - Images
    - Documents
    - Videos
    - Zip Files
    - Settings files
    - Encrypted Files (encrypted locally)

2. Personal CRM
    - Contacts
    - Interaction Log
    - Reminders
    - Calendar

3. Bookmarks collection
   - Organize by Category and Tags

4. InstaSlop
    - A faxu social media with Post generated by AI using the personal Image collection
   
5. Habit's Tracker
6. Notes
7. Personal Journal
8. Gallery
9. News Feed
10. Tip Rounder
11. Medicine Tracker
12. Budgeting
